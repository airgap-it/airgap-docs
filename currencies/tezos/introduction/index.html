Introductionâ€‹

Tezos is a decentralized open-source blockchain that goes beyond peer-to-peer transactions and serves as a programmable platform for smart contracts. The native currency for the Tezos blockchain is tez which has the symbol XTZ. learn more.

AirGap Tezos Walletâ€‹

AirGap Tezos Wallet is a convenient way to store and transfer tez, NFTs and FA1.2 and FA2 tokens while storing your private key offline. The wallet also allows you to interact with FA1.2 and 2 tokens and NFTs. You can securely interact with the decentralized applications built on Tezos using our Beacon functionality.
