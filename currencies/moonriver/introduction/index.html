Moonriver Introduction

Introduction to Moonriverâ€‹

Moonriver is a companion network to Moonbeam and serves as a canary network. It is a layer 1 solution built on Kusama and serves as a testing environment. New code is first deployed to Moonriver, where it can be tested and validated in real-world economic conditions. The same code is then deployed to Moonbeam on Polkadot.

MOVR is the utility token of the Moonriver Network, and it serves the following functions:

  • Payment for transactions fees.
  • Payment for the execution of smart contracts.
  • Incentivizing collators to produce blocks to keep the network running efficiently.
  • Facilitation of Moonriver's on-chain governance system.
