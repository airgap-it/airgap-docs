## Introduction to BNB Chain

BNB Chain (formerly Binance Smart Chain) is an EVM-compatible blockchain ecosystem built by Binance. It consists of BNB Beacon Chain for governance and staking, and BNB Smart Chain (BSC) for smart contract execution. BNB Chain is designed for high throughput and low transaction fees, making it one of the most widely used blockchain networks for DeFi, NFTs, and decentralized applications.

BNB is the native token of the BNB Chain ecosystem. BNB has several functions:

- Paying transaction fees (gas) on the network
- Staking for network security and governance
- Participating in token launches on Binance Launchpad

[Learn more](https://www.bnbchain.org/)

The AirGap BNB Chain Wallet provides a user-friendly UI that allows you to interact with the BNB Smart Chain. It provides a secure and intuitive interface for managing your assets and executing transactions on BSC.
