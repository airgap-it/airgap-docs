## Introduction to Base

Base is a layer 2 scaling solution for the Ethereum blockchain built by Coinbase. It leverages Optimistic Rollup technology to provide fast, low-cost transactions while inheriting the security of Ethereum. Base is built on the OP Stack in collaboration with Optimism, making it part of the Superchain ecosystem.

Base aims to bring the next billion users on-chain by offering a developer-friendly, secure, and low-cost environment for building decentralized applications. It is fully EVM-compatible, meaning existing Ethereum smart contracts and tools work seamlessly on Base.

[Learn more](https://base.org/)

### AirGap Base Wallet

The AirGap Base Wallet provides a user-friendly UI that allows you to interact with the Base blockchain. It provides a secure and intuitive interface for managing your assets and executing transactions on Base.

With the AirGap Base Wallet, you can securely store and transfer ETH and ERC-20 tokens on the Base network. The AirGap Vault also integrates with popular wallets like MetaMask & Rabby Wallet, allowing you to easily connect and manage your Base assets with multiple options. You can securely interact with decentralized applications built on Base using the WalletConnect functionality in the AirGap Wallet or by connecting the AirGap Vault to MetaMask or Rabby Wallet.
